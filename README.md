# SiteSurveyor

Professional Free Geomatics software for Windows, Linux, and macOS.

**Official Website**: [sitesurveyor.dev](https://sitesurveyor.dev)

## Features
- Import/Export (DXF, Shapefile, CSV, GeoJSON)
- COGO Calculations (Polars, Joins, Intersections)
- Network Adjustments (GNU Gama)
- Volumetric Calculations
- Cross-platform support
