# Changelog

All notable changes to SiteSurveyor will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Comprehensive README documentation
- CONTRIBUTING.md with developer guidelines
- Updated issue templates for bug reports and feature requests

---

## [1.0.8] - 2025-12-21

### Added
- Cloud project synchronization with Appwrite backend
- Version history for cloud projects
- Project sharing with team members
- Conflict detection for concurrent edits
- Dark and light theme support
- Animated zoom and pan transitions
- Command console for scripting

### Changed
- Improved traverse calculation dialog with Bowditch and Transit adjustments
- Enhanced volume calculation with TIN viewer
- Updated Qt Advanced Docking System integration

### Fixed
- Layer visibility state persistence
- Coordinate transformation edge cases
- Snap tolerance calculation at different zoom levels

---

## [1.0.7] - 2025-11-15

### Added
- GNU Gama network adjustment integration
- Network adjustment dialog with results display
- Chi-square test and residual analysis
- Export adjusted coordinates to canvas

### Changed
- Improved GDAL/GEOS integration
- Better error handling for file imports

---

## [1.0.6] - 2025-10-20

### Added
- Five survey categories: Engineering, Cadastral, Mining, Topographic, Geodetic
- Category-specific tools and default layers
- Category manager for feature filtering

### Changed
- Menu system now filters based on selected category
- Toolbar updates with category-relevant tools

---

## [1.0.5] - 2025-09-10

### Added
- Traverse calculation dialog
- Forward and backward traverse processing
- Closure error computation
- CSV export for traverse reports

### Fixed
- Bearing calculation precision issues
- DMS formatting edge cases

---

## [1.0.4] - 2025-08-01

### Added
- Volume calculation with TIN generation
- Contour generation from point data
- 3D TIN viewer

### Changed
- Improved point interpolation algorithms

---

## [1.0.3] - 2025-07-01

### Added
- Levelling dialog for differential levelling
- Rise and fall calculation method
- Height of collimation method

---

## [1.0.2] - 2025-06-01

### Added
- Resection calculation from known points
- Intersection calculations (bearing-bearing, bearing-distance)
- Polar and join calculations

---

## [1.0.1] - 2025-05-01

### Added
- Authentication system with login dialog
- User profile management
- License validation

### Fixed
- Startup dialog layout issues
- Recent projects list persistence

---

## [1.0.0] - 2025-04-01

### Added
- Initial release
- DXF import/export
- Shapefile import/export via GDAL
- GeoJSON and CSV support
- Basic canvas with layer management
- Point, line, polyline, circle, arc drawing tools
- Snapping functionality
- Undo/redo system
- Peg coordinate management
- Station setup for total station workflows
- Cross-platform support (Windows, Linux, macOS)

---
