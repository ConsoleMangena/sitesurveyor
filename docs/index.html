Drawing & selection

  • Line, polyline, rectangle, circle, arc
  • Select, pan, zoom window, lasso; zoom in/out, extents, home
  • Dynamic input for distance/angle while drawing

Coordinates & data

  • Add points; import/export CSV (name, X/Y/Z); clear project
  • Points table with per-point layer assignment
  • Center view on selected point

Layers

  • Create/rename/remove; set color, visible, locked
  • Set current layer; hide/isolate/lock selected layers; show all

Survey tools

  • Distance, area, azimuth between named points
  • Polar input (from point + distance + azimuth), optional draw line
  • Join (polar) between two points; Traverse with multiple legs

Aids & display

  • Grid, labels, crosshair; ortho, snap, object snap, polar tracking, otrack
  • Zoom percentage and live drawing distance readout

UI & panels

  • Dockable panels: points, layers, properties, command line, project plan
  • Toggle left/right/command panels; reset layout; dark mode
  • Settings for units (metric/imperial), angle format (DMS/decimal), Gauss order, 2D/3D

Install

Windows

  1. Download the ZIP above.
  2. Extract it, then run SiteSurveyor.exe.
  3. If SmartScreen warns, choose More info → Run anyway.

Debian/Ubuntu

  1. Download the .deb above.
  2. Install with sudo dpkg -i SiteSurveyor-*.deb.
  3. Launch “SiteSurveyor Desktop” from your menu.

Source code and build instructions are on GitHub.
