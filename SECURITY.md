# Security Policy

## Supported Versions

The following versions of SiteSurveyor are currently receiving security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability in SiteSurveyor, please report it responsibly.

### How to Report

1. **Do NOT** open a public GitHub issue for security vulnerabilities
2. Email security concerns to: **security@sitesurveyor.dev**
3. Include the following information:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

### What to Expect

- **Acknowledgment**: We will acknowledge receipt within 48 hours
- **Assessment**: We will assess the vulnerability and determine its severity
- **Timeline**: We aim to address critical vulnerabilities within 7 days
- **Updates**: We will keep you informed of our progress
- **Credit**: With your permission, we will credit you in the security advisory

### Scope

Security vulnerabilities we are interested in include:

- Authentication bypass
- Data exposure or leakage
- Cloud sync security issues
- Local privilege escalation
- Arbitrary code execution
- Cross-site scripting (if web components are affected)

### Out of Scope

The following are generally not considered security vulnerabilities:

- Denial of service through resource exhaustion on local machine
- Issues requiring physical access to the device
- Social engineering attacks
- Issues in third-party dependencies (report to the appropriate project)

## Security Best Practices

When using SiteSurveyor:

1. **Keep Updated**: Always use the latest version
2. **Secure Credentials**: Protect your cloud account credentials
3. **Trusted Sources**: Only download from official sources
4. **File Sources**: Be cautious when opening project files from untrusted sources

## Acknowledgments

We thank the security research community for helping keep SiteSurveyor secure.
